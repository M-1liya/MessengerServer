Кодовые слова и их очередность.

HEAR_ME?		- Проверка связи с клиентским приложением, ответить нужно что угодно кроме пустого сообщения
LOG_IN			- Команда для авторизации. После отправки этой команды необходимо следом отправить данные для входа.
AUTH			- Команда для регистрации. После отправки этой команды необходимо следом отправить данные для регистрации.
SEND			- Команда для отправки сообщения пользователю. Формат команды: SEND::<id>::<message>.
WRONG_TEMPLATE	- Данное сообщение отправляется в случае нарушения формата каманды SEND.
INVALID_REQUEST	- Такой ответ от сервера говорит о том, что полученный запрос не соответствует ни одной команде.
